# Maven Movies: Insurance Underwriting Analysis

## Project Context
The business information for Maven Movies hasn't been updated in years. To approve a policy renewal, Lead Underwriter **Joe Scardycat** requested an extraction of key metrics to assess operational risk.

## Tools Used
* **Database:** MySQL
* **Key Skills:** Data Aggregation, Risk Assessment, Business Intelligence Reporting

## Analysis Highlights
The included SQL script provides answers to 8 critical business questions, including:
* **Asset Valuation:** Calculating the replacement cost of the entire film library.
* **Liability Tracking:** Counting customer email entries to assess data breach exposure.
* **Fraud Prevention:** Monitoring average and maximum payment amounts.
* **Inventory Management:** Comparing stock levels across different store locations.
